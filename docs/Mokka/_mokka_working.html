Working with Mokka

The Mokka user interface is designed to have both access and visualization of all the data included in a particular motion study.

It is divided in 4 mains parts:

  • Menu Bar
  • Acquisition explorer
  • Time Bar
  • Views (3D views, Charts, Media, Console)

You have also access to preferences to select / modify / add / remove default configurations in Mokka.
