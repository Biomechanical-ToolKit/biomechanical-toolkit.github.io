Detailed Description

Chart for the time series data with only a bottom and left axes.

This chart has several interaction to constraint pan and zoom

  • Select only a horizontal or vertical zoom;
  • Pan the chart only into boundaries;
  • Zoom box activated by the key SHIFT.

You can also enable/disable the interactions .

You have also the possibilty to display the current frame and the region of interest of the time series, as well as events. To be able to display these informations, you have to create functors inheriting from VTKCurrentFrameFunctor, VTKRegionOfInterestFunctor and VTKEventsFunctor. Then set the functor by using the methods SetCurrentFrameFunctor(), SetRegionOfInterestFunctor() and SetEventsFunctor.

The events are optional and hidden by default (and there is no functor). To display them, and after setting the functor, you have to use the function DisplayEventsOn().
