template<typename T >
void btk::Variant::SetValue ( int  idx,
const T &  val 
)

Sets val as the value for the given index idx. If necessary, the value is adapted to fit the format.

In case the format of the variant is a string, the dimension can be adapted to fit with the size of the given (stringified) value.

If the given index is out of range, the method send an error message to the logger.
