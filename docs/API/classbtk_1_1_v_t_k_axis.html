Detailed Description

Class to display axis into the chart with ticks drawn inside the chart.

The direction of the ticks can be set with the method SetTickDirection(). This class take into account the transformation given to the object to paint correctly the legend into the scene (only the scale and translation are supported).

The value for the ticks' label can be influenced by an offset and a scale using the following operation: tick_label = (real_value + offset) * scale. To set the scale and the offset, you have to use the methods SetTickScale() and SetTickOffset() respectively
