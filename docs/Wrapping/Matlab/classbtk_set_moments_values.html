btkSetMomentsValues Class Reference
Modify point data

Description

Sets moments' components from a matrix.

Syntax:
btkSetMomentsValues(h, v)
Parameters
hHandle pointing to a C++ btk::Acquisition object.
vMatrix of reals where the number of rows corresponds to the number of frames in the acquisition. The number of columns corresponds to the number of extracted moments multiplied by 3 (their components X,Y,Z).
